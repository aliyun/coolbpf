# eBPF成为趋势

eBPF基本概念介绍,以及eBPF发展历史
eBPF与内核模块优劣势比较及分析
它无需更改内核源代码或加载内核模块，方便的内核可编程性，使基础架构软件可以利用现有的层，从而使它们更加智能和功能丰富，而无需继续为系统增加额外的复杂性层，eBPF将成为实现Linux内核多样性和自定义的一种最优解决方案。
